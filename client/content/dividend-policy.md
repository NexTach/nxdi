# NexTach Global Dividend Income 분배 정책

[작성일] 2026/07/15

## 1. 의향서와 분배권의 구분

사이트에서 제출하거나 회사가 `수락`으로 표시한 투자 의향서는 계약이 아닙니다. 의향서 금액은 실제 투자원금이나 분배원금에 포함되지 않으며, 의향서 제출자에게 분배금·재투자금·출금권을 발생시키지 않습니다.

계약 체결과 지정 계좌 입금만으로는 분배원금이 생기지 않습니다. 실제 매수 체결로 구성종목에 편입되고 투자자 귀속 취득금액이 운용편입 원장에 기록된 금액만 분배 대상입니다. 첫 분배 대상월은 운용 편입이 확정된 달의 다음 달입니다. 출금은 실제 지급이 확정된 금액을 원장에서 차감합니다.

수락된 투자·출금 의향의 총액과 이를 이용한 사이트 계산기는 연락 및 내부 참고용으로 유지합니다. 이 값은 실제 포트폴리오 자산총액, 운용편입 원금, 손실흡수 순서 또는 지급액을 변경하지 않습니다.

## 2. 정책값

| 항목         |                    기준 |
|------------|----------------------:|
| 회사 분배금 이전율 |        회사 기본몫의 20% 이내 |
| 월 현금지급 상한  | 지급월 투자자 운용편입 원금의 1.5% |
| 분배연동 운용보수  |          투자자 총귀속액의 5% |
| 현금지급 예정일   |  매월 말일, 비영업일이면 다음 영업일 |
| 환헤지        |                   미실시 |

월 1.5%를 12배 한 18%는 단순 연환산 참고치이며 연간 누적 상한, 목표수익률 또는 보장수익률이 아닙니다. 배분가능 현금이 없으면 분배금과 운용보수는 모두 0원입니다.

## 3. 월 전체 실분배금의 정의

각 기초자산의 USD 현금분배금은 실제 입금일의 증권사 체결환율로 즉시 원화 환전합니다. `월 전체 실분배금`은 다음 조정 뒤 분리계좌에 실제 정산된 원화 순입금액입니다.

```text
기초자산 총 현금분배금
- 해외 원천세
- 증권사·거래소 비용
- 환전 스프레드와 환전비용
= 월 전체 실분배금 A
```

배당, 이자, 자본이득 분배 및 자본환급 성격은 확인 가능한 범위에서 구분합니다. 투자자별 국내 원천징수 등은 아래 산식 이후 개별 명세에서 별도로 공제합니다.

각 입금 건은 증권사 거래명세 식별값을 중복방지 키로 사용하여 종목·통화·외화 총액·실제 체결환율·원화 총액·외국세·증권사비용·환전비용·원화 순입금액·입금시각을 기록합니다. 월 전체 실분배금은 해당 월의 유효한 순입금 원장을 자동 합산한 값만 사용하며 임의 합계나 대체환율을 허용하지 않습니다. 입력 오류는 삭제하지 않고 사유를 남긴 반대분개로 정정합니다.

## 4. 월별 산정 순서

지급월 기준 전체 투자자 운용편입 원금을 `P`, 월말 확정 포트폴리오 순자산을 `V`, 월 전체 실분배금을 `A`라고 합니다.

```text
회사 기준금액 C = max(V - P, 0)

투자자 기본몫 = A x P / (P + C)
회사 기본몫 = A - 투자자 기본몫

월 현금상한 = P x 1.5%
회사 이전액 = min(
  max(월 현금상한 - 투자자 기본몫, 0),
  회사 기본몫 x 20%,
  회사 기본몫
)

투자자 총귀속액 = 투자자 기본몫 + 회사 이전액
운용보수 = 투자자 총귀속액 x 5%
투자자 순귀속액 = 투자자 총귀속액 - 운용보수

현금지급 총액 = min(투자자 순귀속액, 월 현금상한)
배당 재투자금 = max(투자자 순귀속액 - 현금지급 총액, 0)
회사 귀속액 = A - 현금지급 총액 - 배당 재투자금
```

현금지급 총액, 배당 재투자금 및 운용보수는 각 투자자의 운용편입 원금 비율로 나눕니다. 각 투자자 배정액은 먼저 원 단위로 내린 뒤 소수점 잔여가 큰 순서로 1원씩 배정하는 최대잔여법을 사용하며, 동률이면 내부 사용자 식별값 순으로 처리합니다. 월 전체 산식에서 생긴 원 단위 잔액은 투자자가 한 명이라도 있으면 재투자금에 더하고 최대잔여법으로 배분하므로 회사 수익이나 설명되지 않은 잔액으로 남기지 않습니다.

## 5. 회사의 20% 이전

투자자 기본몫이 월 현금상한보다 적을 때 회사 기본몫의 최대 20%를 투자자 총귀속액에 더합니다. 이는 회사가 자기 몫 일부를 포기하는 투자자 보호장치입니다.

다만 실제 회사 기본몫이 있어야 하고, 부족분·회사 기본몫의 20%·회사 기본몫 중 가장 작은 금액만 이전합니다. 따라서 월 1.5%를 채우는 보증, 손실보전 약정 또는 확정수익 약정이 아닙니다.

## 6. 운용보수

운용보수는 `회사 이전액을 포함한 투자자 총귀속액 × 5%`입니다. 산정 기준일은 매월 말일이며 분배 정산과 동시에 회사 귀속액으로 지급합니다.

- 투자자 총귀속액이 0원이면 보수도 0원입니다.
- 보수는 투자원금에서 차감하지 않습니다.
- 해당 월에 발생하지 않은 보수를 다음 달로 이월하지 않습니다.
- 외부 거래·환전·보관 비용과 투자자별 법정 세금은 운용보수와 별도입니다.
- 운용보수의 산정 기준, 요율 또는 지급시점을 기존 계약에 불리하게 소급 변경하지 않습니다.

## 7. 현금 상한 초과분의 재투자

운용보수 차감 후 투자자 순귀속액이 월 현금상한을 넘으면 초과분은 회사에 귀속되지 않습니다. 투자자별 원금비율로 나누어 `배당 재투자금`으로 확정합니다.

초과분은 말일 정산 후 `재투자 대기금`으로 확정합니다. 대기금은 현금성 투자자 귀속액이지만 아직 운용편입 원금이 아닙니다. 해당 금액으로 실제 매수가 체결된 날 투자자 귀속 취득금액만 운용편입 원금으로 전환합니다. 국내 원천징수 등 법정 공제가 필요한 경우 순재투자 대기금은 세액 공제 후 금액입니다. 투자자는 현금으로 받지 않은 금액에도 과세가 발생할 수 있으므로 개별 명세를 확인해야 합니다.

## 8. 예시

투자자 운용편입 원금 `P=1,000,000원`, 월말 순자산 `V=1,250,000원`, 월 실분배금 `A=20,000원`인 경우입니다.

| 항목               |      금액 |
|------------------|--------:|
| 투자자 기본몫          | 16,000원 |
| 회사 기본몫           |  4,000원 |
| 회사 이전액           |      0원 |
| 투자자 총귀속액         | 16,000원 |
| 운용보수 5%          |    800원 |
| 투자자 순귀속액         | 15,200원 |
| 현금지급 총액(1.5% 상한) | 15,000원 |
| 배당 재투자금          |    200원 |
| 회사 귀속액           |  4,800원 |

`15,000 + 200 + 4,800 = 20,000원`으로 월 실분배금과 일치합니다. 투자원금 100,000원인 투자자는 원금비율 10%에 따라 세금 공제 전 현금지급액 1,500원과 재투자금 20원을 배정받습니다.

## 9. 지급과 정산

지급 예정일은 매월 말일이며 말일이 비영업일이면 다음 영업일에 지급합니다. 증권사 결제 지연, 시장·은행 휴무, 거래정지, 계좌 오류, 법정 확인절차 또는 검증자료 부재가 있으면 지급이 지연될 수 있습니다.

월 정산은 해당 월의 종목별 순입금 원장, 실제 운용편입 원금, 확정 월말 순자산 및 세무 검토로 확인한 원천징수율이 모두 있어야 계산합니다. 원천징수율은 임의 기본값을 사용하지 않습니다. 계산된 정산안은 지급 전 검토단계이며, 정산안 계산 뒤 입금 원장이 달라지면 확정을 자동 중단하고 재계산합니다. 확정된 월 정산은 삭제하거나 다시 계산하지 않고, 오류는 별도 정정 원장과 다음 지급명세로 처리합니다.

월말에는 다음 항목을 확정하여 개별 명세를 제공합니다.

- 계약금·입금액·미편입 예수금과 운용편입 원금의 변동내역
- 월말 순자산과 회사 잔여 손실흡수 자본
- 총 현금분배금, 외국세금, 환전·외부비용, 월 실분배금
- 투자자 기본몫, 회사 이전액, 운용보수
- 현금지급액, 재투자금, 투자자별 법정 세금

검증된 환율이나 평가정보를 확보하지 못하면 임의 대체값을 사용하지 않고 산정·승인을 자동 중단합니다.

현금지급이 계좌 오류 등으로 실패하면 해당 금액은 투자자 미지급금으로 남기고 회사 귀속액으로 전환하지 않습니다. 본인명의 계좌를 다시 확인한 후 재지급하며, 압류·사망·상속분쟁·법정 제한이 확인되면 권한관계가 확인될 때까지 지급을 보류합니다.

## 10. 평가손익과 원금손실

투자자는 평가이익에 참여하지 않으므로 자산가격이나 환율이 상승해도 운용편입 원금은 늘지 않습니다. 의향 수락, 계약, 입금 또는 재투자 대기만으로도 원금은 늘지 않습니다. 반면 손실은 회사 손실흡수 자본이 먼저 부담하고, 월말 순자산이 전체 투자자 운용편입 원금보다 낮아진 뒤에만 투자자 출금 기준액에 반영됩니다.

```text
투자자 손실률 = min(max((월말 순자산 - 전체 투자자 운용편입 원금)
                         / 전체 투자자 운용편입 원금, -100%), 0%)
```

회사 자본은 투자자 손실 가능성을 낮추는 완충장치일 뿐 원금보증이 아닙니다. 회사 자본을 넘는 손실, 환율 변동, 분배금 감소, 세금·비용 및 유동성 부족으로 원금 손실이나 지급 지연이 발생할 수 있습니다.
