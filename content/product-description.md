# T-ETF 상품 설명

T-ETF(이하 당주)는 유가증권시장

## 핵심 구조

| 항목 | 설명 |
| --- | --- |
| 기준 포트폴리오 | 관리자가 등록한 보유 종목, 수량, 환율, 배당 데이터를 기준으로 표시합니다. |
| 투자 의향 | 사용자가 참여 희망 금액과 연락처를 제출하면 관리자가 상태를 검토합니다. |
| 출금 의향 | 수락된 투자 원금과 포트폴리오 하락률을 반영한 출금 가능 상한을 안내합니다. |
| 배당 전망 | 현재 보유 수량 기준으로 예정 배당을 월별 또는 종목별로 보여줍니다. |

## 사용자가 확인할 수 있는 것

- 포트폴리오 평가금액과 구성 종목 비중
- 오늘 등락률, 보유 수익률, 배당수익률
- 종목별 가격 흐름과 배당 정보
- 가정 투자금 기준 예상 배당 시뮬레이션
- 본인이 제출한 투자 의향과 출금 의향 상태

## 운영 원칙

1. 포트폴리오와 배당 데이터는 관리자가 수동으로 관리합니다.
2. 제출된 의향서는 확정 계약이 아니며, 관리자의 확인 절차를 거칩니다.
3. 수익률과 배당 전망은 외부 시장 데이터와 입력값에 따라 달라질 수 있습니다.
4. 사용자는 실제 송금, 환불, 계약 조건을 서비스 외부에서 별도로 확인해야 합니다.

## 유의사항

이 페이지의 내용은 상품 설명을 돕기 위한 안내입니다. 투자 판단의 책임은 사용자에게 있으며, 표시된 데이터는 지연되거나 실제 체결 조건과 다를 수 있습니다.
