# NexTach Global Dividend Income Fund 배당 정책

이 문서는 상품 설명의 "배당 산정 정책"을 자세히 풀어 쓴 해설 자료입니다. 상품 설명이 배당 정책의 요약이라면, 이 문서는 그 정책이 어떤 원칙 위에서, 어떤 순서로, 왜 그렇게 계산되는지를 단계별 예시와 함께 설명합니다. 계산 공식 자체는 상품 설명의 "월별 투자자 배당 배분 공식" 및 서비스의 배당 배분 계산 로직과 동일합니다.

먼저 전제 하나를 분명히 합니다. NexTach Global Dividend Income Fund(티커 "TDIV", 국문명 "넥스태치 글로벌 배당 인컴 펀드", 이하 "TDIV")의 월 배당은 정해진 이자가 아닙니다. 매월 지급되는 배당 금액은 그 달에 포트폴리오에서 실제로 발생한 배당금에 따라 달라지며, 어떤 달에는 목표 상한에 도달하고 어떤 달에는 그보다 적을 수 있습니다. 이 문서의 모든 계산은 "그 달에 실제 들어온 배당금 안에서" 이루어집니다.

## 적용 대상

배당 배분은 관리자가 수락한 투자 의향서의 투자 원금을 기준으로 계산합니다. 대기 중이거나 거절된 투자 의향서는 배당 배분 대상 원금에 포함하지 않습니다.

투자자가 제출한 투자 의향서는 확정 계약이 아니며, 관리자의 수락과 실제 입금 확인 절차를 거친 뒤 배당 배분 대상에 포함됩니다.

## 1. 배당 정책의 5가지 기본 원칙

배당 정책은 다음 다섯 가지 원칙 위에서 설계되었습니다. 뒤의 계산 공식은 이 원칙들을 숫자로 옮긴 것입니다.

**1. 실배당 기반 원칙**

분배는 그 달에 실제로 발생한 배당금 안에서만 이루어집니다. 실제 배당이 적으면 지급도 적어지며, 없는 배당을 만들어 지급하지 않습니다.

**2. 원금 비율 배분 원칙**

투자자 사이의 배분 비율은 포트폴리오 평가금액이 아니라, 각 투자자의 승인된 투자 원금이 전체 승인 투자 원금에서 차지하는 비율로 정합니다. 평가이익이나 손실이 개별 투자자의 몫 비율을 바꾸지 않습니다.

**3. 증자금 비희석 원칙**

당사 증자금은 투자자 사이의 배분 비율을 희석하지 않습니다. 당사가 자금을 더 넣는다고 해서 개별 투자자가 받을 몫의 비율이 줄어들지 않습니다.

**4. 월 상한 원칙**

투자자에게 지급되는 전체 배당은 매월 정해진 상한, 즉 투자자 원금 합계의 0.25%를 넘지 않습니다. 실배당이 아무리 많아도 투자자 배당은 이 상한에서 멈추고, 초과분은 당사가 보유합니다.

**5. 당사 몫 이전 원칙**

투자자 기본배당이 월 상한에 못 미치는 경우에는 당사 몫으로 산정된 배당의 일부, 최대 20%를 투자자 쪽으로 이전해 배당을 보완할 수 있습니다. 다만 이전 역시 실제 발생한 배당 안에서만 이루어지며, 없는 배당을 채워주지는 않습니다.

## 2. 용어 정의

| 용어 | 의미 |
|------|------|
| 투자자 원금 합계 | 승인된 전체 투자자 투자 원금의 합 |
| 포트폴리오 평가금액 | 월말 기준 포트폴리오 전체의 평가금액 |
| 월 전체 실배당금 | 해당 월에 실제로 입금된 총 배당금 |
| 당사 기준금액 | 평가금액에서 투자자 원금을 뺀 부분. 음수면 0으로 계산 |
| 전체 기준금액 | 투자자 원금 합계 + 당사 기준금액 |
| 투자자 기본배당 | 원금 비율에 따라 투자자에게 우선 귀속되는 배당 |
| 당사 기본배당 | 전체 실배당에서 투자자 기본배당을 뺀 나머지 |
| 월 상한금액 | 투자자 원금 합계 x 0.25%. 투자자 전체 배당의 월 상한 |
| 당사 이전액 | 상한에 못 미칠 때 당사 몫에서 투자자로 이전하는 금액 |
| 투자자 배분대상 | 최종적으로 투자자 전체에게 배분되는 배당 총액 |
| 개별 지급액 | 각 투자자가 실제 받는 금액 |
| 당사 보유배당 | 투자자에게 배분하고 남은, 당사가 보유하는 배당 |

## 3. 배당 계산 단계별 설명

한 달의 배당은 다음 다섯 단계로 계산됩니다. 각 단계가 앞의 어떤 원칙을 구현하는지 함께 표시했습니다.

### 1단계 - 기준금액 산정

원칙 2, 3을 반영합니다.

```
당사 기준금액 = max(포트폴리오 평가금액 - 투자자 원금 합계, 0)
전체 기준금액 = 투자자 원금 합계 + 당사 기준금액
```

포트폴리오 평가금액에서 투자자 원금을 뺀 부분을 당사 배당 몫을 산정하기 위한 기준금액으로 봅니다. 평가금액이 원금보다 낮은 하락 구간에서는 당사 기준금액이 0이 되어, 그 달 배당은 전부 투자자 몫으로 귀속됩니다.

### 2단계 - 기본 배당 몫 나누기

원칙 1, 2를 반영합니다.

```
투자자 기본배당 = 월 전체 실배당금 x 투자자 원금 합계 / 전체 기준금액
당사 기본배당 = 월 전체 실배당금 - 투자자 기본배당
```

실제 발생한 배당금을 기준금액 비율에 따라 투자자 몫과 당사 몫으로 나눕니다. 투자자 몫은 "투자자 원금 / 전체 기준금액" 비율만큼입니다.

### 3단계 - 당사 몫 배당 이전

원칙 4, 5를 반영합니다.

```
월 상한금액 = 투자자 원금 합계 x 0.25%
부족분 = max(월 상한금액 - 투자자 기본배당, 0)
당사 이전액 = min(부족분, 당사 기본배당 x 20%, 당사 기본배당)
```

투자자 기본배당이 월 상한에 못 미치면, 그 부족분을 당사 몫에서 보완할 수 있습니다. 단 이전액은 당사 기본배당의 20%를 넘을 수 없고, 당사 기본배당 자체를 넘을 수도 없습니다. 즉 세 값 중 가장 작은 값만큼만 이전됩니다.

### 4단계 - 최종 배분 확정

원칙 1, 4를 반영합니다.

```
투자자 배분대상 = min(투자자 기본배당 + 당사 이전액, 월 상한금액, 월 전체 실배당금)
당사 보유배당 = 월 전체 실배당금 - 투자자 배분대상
```

투자자 전체 배당은 "기본배당 + 이전액", "월 상한", "실제 발생 배당" 셋 중 가장 작은 값으로 확정됩니다. 이 min 구조가 상한 초과와 실배당 초과를 동시에 막습니다.

### 5단계 - 개별 지급액 계산

원칙 2, 3을 반영합니다.

```
개별 지급액 = 투자자 배분대상 x 개별 투자원금 / 투자자 원금 합계
```

확정된 투자자 배분대상을 각자의 원금 비율대로 나눕니다. 개별 투자자의 몫은 자신의 원금 비율로 결정되며, 당사 증자금은 투자자 사이의 배분 비율을 바꾸지 않습니다.

## 4. 공식 요약

```dividend-formula
title: 월별 투자자 배당 배분 공식
description: 실제 배당금 안에서 계산하며, 투자자 전체 배당은 월 상한을 넘지 않습니다.
# 상품 정책값
당사 배당 이전율 = 20%
월 투자자 배당 상한율 = 0.25%

# 입력값
투자자 원금 합계 = 승인된 투자 원금 전체
포트폴리오 평가금액 = 월말 포트폴리오 평가금액
월 전체 실배당금 = 해당 월 실제 입금된 총 배당금

# 기준금액
당사 기준금액 =
  max(포트폴리오 평가금액 - 투자자 원금 합계, 0원)

전체 기준금액 =
  투자자 원금 합계 + 당사 기준금액

# 기본 배당 몫
투자자 기본배당 =
  월 전체 실배당금 x 투자자 원금 합계 / 전체 기준금액

당사 기본배당 =
  월 전체 실배당금 - 투자자 기본배당

# 당사 몫 배당 이전
월 상한금액 =
  투자자 원금 합계 x 0.25%

부족분 =
  max(월 상한금액 - 투자자 기본배당, 0원)

당사 이전액 =
  min(부족분, 당사 기본배당 x 20%, 당사 기본배당)

# 최종 배분
투자자 배분대상 =
  min(투자자 기본배당 + 당사 이전액, 월 상한금액, 월 전체 실배당금)

개별 지급액 =
  투자자 배분대상 x 개별 투자원금 / 투자자 원금 합계

당사 보유배당 =
  월 전체 실배당금 - 투자자 배분대상
```

## 5. 정책값 설명

정책값은 두 가지이며, 당사가 매월 임의로 바꾸지 않고 변경이 필요한 경우 상품 설명과 공시를 통해 공개합니다.

**월 투자자 배당 상한율 0.25%**

투자자 전체가 한 달에 받을 수 있는 배당의 상한을 투자자 원금 합계의 0.25%로 둡니다. 예를 들어 투자자 원금 합계가 100만 원이면 그 달 투자자 배당 상한은 2,500원입니다. 월 0.25%는 단순 환산 시 연 약 3% 수준의 목표 상한에 해당합니다. 이는 "목표 상한"일 뿐이며, 실배당이 부족하면 이보다 적게 지급됩니다.

**당사 배당 이전율 20%**

투자자 기본배당이 월 상한에 못 미칠 때, 당사 몫으로 산정된 배당 중 최대 20%까지만 투자자 쪽으로 이전합니다. 당사 몫 전부를 넘겨 투자자 배당을 무리하게 채우지 않기 위한 제한입니다.

## 6. 계산 예시

아래 예시는 모두 투자자 원금 합계 100만 원을 기준으로 하며, 공식에 실제로 대입해 계산한 값입니다. 원 단위 미만은 반올림했습니다.

### 예시 A - 일반적인 달

입력: 원금 합계 100만 원, 평가금액 125만 원, 월 실배당 2,000원

| 단계 | 계산 | 결과 |
|------|------|------|
| 당사 기준금액 | max(125만 - 100만, 0) | 25만 원 |
| 전체 기준금액 | 100만 + 25만 | 125만 원 |
| 투자자 기본배당 | 2,000 x 100만 / 125만 | 1,600원 |
| 당사 기본배당 | 2,000 - 1,600 | 400원 |
| 월 상한금액 | 100만 x 0.25% | 2,500원 |
| 부족분 | max(2,500 - 1,600, 0) | 900원 |
| 당사 이전액 | min(900, 400 x 20% = 80, 400) | 80원 |
| 투자자 배분대상 | min(1,600 + 80, 2,500, 2,000) | 1,680원 |
| 당사 보유배당 | 2,000 - 1,680 | 320원 |

투자자 기본배당 1,600원이 상한 2,500원에 못 미쳐 당사 몫에서 80원을 이전했습니다. 이전액은 부족분 900원이 아니라 당사 기본배당의 20%인 80원으로 제한됩니다. 원금 10만 원 투자자의 개별 지급액은 1,680 x 10만 / 100만 = **168원**입니다.

### 예시 B - 실배당이 매우 적은 달

입력: 원금 합계 100만 원, 평가금액 125만 원, 월 실배당 500원

| 단계 | 계산 | 결과 |
|------|------|------|
| 전체 기준금액 | 100만 + 25만 | 125만 원 |
| 투자자 기본배당 | 500 x 100만 / 125만 | 400원 |
| 당사 기본배당 | 500 - 400 | 100원 |
| 월 상한금액 | 100만 x 0.25% | 2,500원 |
| 당사 이전액 | min(2,100, 100 x 20% = 20, 100) | 20원 |
| 투자자 배분대상 | min(400 + 20, 2,500, 500) | 420원 |
| 당사 보유배당 | 500 - 420 | 80원 |

상한은 2,500원이지만, 그 달 실배당이 500원뿐이므로 투자자 배당은 420원에 그칩니다. 실배당이 부족하면 상한보다 적게 지급된다는 원칙이 그대로 나타나는 경우입니다.

### 예시 C - 실배당이 풍부한 달

입력: 원금 합계 100만 원, 평가금액 125만 원, 월 실배당 5,000원

| 단계 | 계산 | 결과 |
|------|------|------|
| 전체 기준금액 | 100만 + 25만 | 125만 원 |
| 투자자 기본배당 | 5,000 x 100만 / 125만 | 4,000원 |
| 월 상한금액 | 100만 x 0.25% | 2,500원 |
| 당사 이전액 | 기본배당이 상한 초과 -> 이전 없음 | 0원 |
| 투자자 배분대상 | min(4,000, 2,500, 5,000) | 2,500원 |
| 당사 보유배당 | 5,000 - 2,500 | 2,500원 |

투자자 기본배당 4,000원이 상한 2,500원을 넘어, 상한에서 멈춥니다. 초과분은 당사가 보유합니다. 상한이 투자자 배당의 과도한 증가를 막는 경우입니다.

### 예시 D - 평가금액이 원금보다 낮은 달

입력: 원금 합계 100만 원, 평가금액 95만 원, 월 실배당 2,000원

| 단계 | 계산 | 결과 |
|------|------|------|
| 당사 기준금액 | max(95만 - 100만, 0) | 0원 |
| 전체 기준금액 | 100만 + 0 | 100만 원 |
| 투자자 기본배당 | 2,000 x 100만 / 100만 | 2,000원 |
| 당사 기본배당 | 2,000 - 2,000 | 0원 |
| 당사 이전액 | 당사 기본배당 0 -> 이전 없음 | 0원 |
| 투자자 배분대상 | min(2,000, 2,500, 2,000) | 2,000원 |
| 당사 보유배당 | 2,000 - 2,000 | 0원 |

평가금액이 원금보다 낮으면 당사 기준금액이 0이 되어, 그 달 배당은 전부 투자자 몫이 됩니다. 하락 구간에서는 당사가 배당 몫을 가져가지 않는 구조입니다.

### 예시 요약

| 상황 | 평가금액 | 실배당 | 투자자 배분 | 당사 보유 | 특징 |
|------|---------|--------|------------|-----------|------|
| A 일반 | 125만 | 2,000 | 1,680 | 320 | 당사 이전 80, 상한 미도달 |
| B 저배당 | 125만 | 500 | 420 | 80 | 상한보다 적게 지급 |
| C 고배당 | 125만 | 5,000 | 2,500 | 2,500 | 상한 작동, 초과분 당사 보유 |
| D 하락 | 95만 | 2,000 | 2,000 | 0 | 당사 몫 0, 전액 투자자 |

네 예시의 투자자 배분액이 모두 다르다는 점이 핵심입니다. 같은 원금이라도 매월 실배당과 평가금액에 따라 지급액이 달라집니다.

## 7. 지급 시점

TDIV는 매월 말일 배당 지급을 목표로 합니다. 첫 배당은 투자일이 속한 달의 다음 달 말일부터 지급하는 것을 원칙으로 합니다. 예를 들어 7월 중 투자한 경우 8월 말일 첫 배당 지급 대상이 됩니다.

배당금은 관련 세금을 원천징수한 후의 금액으로 지급됩니다. 증권사, 거래소, 은행, 환율 처리 일정에 따라 지급 시점은 지연될 수 있습니다.

## 8. 자주 묻는 질문

**Q. 매달 같은 금액을 받나요?**

아닙니다. 지급액은 그 달의 실제 배당금과 포트폴리오 상황에 따라 매월 달라집니다. 0.25%는 받는 금액이 아니라 넘지 않는 상한입니다.

**Q. 상한 0.25%는 항상 받을 수 있나요?**

아닙니다. 상한은 목표치이며, 실배당이 부족하면 예시 B처럼 그보다 적게 지급됩니다.

**Q. 당사가 증자를 많이 하면 제 몫이 줄어드나요?**

아닙니다. 개별 지급액은 자신의 원금 비율로만 정해지므로, 당사 증자금은 투자자 사이의 배분 비율을 바꾸지 않습니다.

**Q. 포트폴리오가 하락하면 배당을 아예 못 받나요?**

평가금액이 원금보다 낮아도 그 달 실배당이 있으면 예시 D처럼 배당은 지급됩니다. 다만 실배당 자체가 줄면 배당액도 줄어듭니다. 배당과 원금 손실은 별개이며, 원금은 보장되지 않습니다.

## 9. 요약 및 유의사항

배당 정책의 골격은 "실제 발생한 배당 안에서, 원금 비율로, 월 상한을 넘지 않게 배분한다"입니다. 상한에 못 미치면 당사 몫의 일부, 최대 20%로 보완하되, 없는 배당을 만들지는 않습니다.

이 문서의 예시 수치는 정책 이해를 돕기 위한 가정값이며, 실제 배당은 시장 상황, 포트폴리오 성과, 환율, 세금, 수수료에 따라 달라집니다. 배당금은 세금 원천징수 후 지급되며, 투자 판단의 책임은 투자자에게 있습니다. 정책값인 이전율 20%와 상한 0.25%가 변경될 경우 상품 설명과 공시를 통해 공개합니다.
